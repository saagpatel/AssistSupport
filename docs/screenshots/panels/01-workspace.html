Can I use a flash drive on my work MacBook for the offsite?

Removable media is permitted on company-issued Macs for short-term travel use, but it must be an IT-approved encrypted drive and the file transfer has to be logged through the macOS Endpoint Security agent. Per IT Security Policy 4.2, unencrypted flash drives are blocked at the kernel extension layer.

For the offsite this week, Priya has two approved options:

The Kingston IronKey D500S 16GB is standard-issue and can be picked up same-day from the 3rd-floor IT desk — it ships FIPS-validated and auto-enrolls with the device-management console. If she already owns an Apricorn Aegis Secure Key 3NX, IT can whitelist its serial in under ten minutes.

Either way, she should request the drive before end of day tomorrow so the endpoint policy has time to propagate to her Mac before departure. Unapproved drives inserted on travel Wi-Fi will trigger an automatic incident in PagerDuty and lock the port for 24 hours.
