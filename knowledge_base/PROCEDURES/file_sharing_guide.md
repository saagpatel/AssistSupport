# File Sharing Guide — Approved Methods

## Internal Sharing (Within Company)

### OneDrive / SharePoint
1. Upload file to OneDrive or SharePoint
2. Click "Share" > "People in [Company] with the link"
3. Set permissions (View only / Edit)
4. Copy link and share via Teams or email

### Microsoft Teams
1. Attach file directly in Teams chat or channel
2. Files are stored in SharePoint automatically
3. All team members can access

## External Sharing (With Partners/Vendors)

### ShareFile (Preferred for Sensitive Data)
1. Log into ShareFile at https://company.sharefile.com
2. Upload file
3. Create a secure link with expiration date
4. Set password protection if required
5. Share link via email

### Dropbox Business
1. Upload to Dropbox Business
2. Share via link with "Anyone with link" or specific email
3. Set expiration and password as needed

### Encrypted Email
1. Compose email in Outlook
2. Click "Encrypt" in the toolbar
3. Attach file (max 25MB)
4. Send — recipient will need to verify identity to open

## NOT Allowed

- Flash drives or USB storage (see Flash Drive Policy)
- Personal cloud storage accounts
- WeTransfer, MediaFire, or other unauthorized services
- Peer-to-peer file sharing
